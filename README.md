# Plataforma de Chamados de TI

Este projeto é uma plataforma online para registro de chamados de TI, desenvolvida para a Prefeitura.

## Tecnologias Utilizadas
- Frontend: Bootstrap, jQuery
- Backend: PHP 8
- Banco de Dados: MySQL

## Funcionalidades
1. Cadastro e login de usuários.
2. Abertura de chamados com anexos e contatos.
3. Visualização de chamados e histórico.
4. Segurança contra SQL Injection e XSS.

## Como Executar
1. Importe o arquivo `database.sql` no MySQL.
2. Configure o XAMPP para rodar o projeto.
3. Acesse `http://localhost/plataforma-chamados-ti`.
